# Hiphop motif pack

Reusable SVG primitives for the `hiphop-stencil-collision` template.

- `sprayburst-red.svg`: impact/punchline burst.
- `faceted-gem.svg`: lyric-specific value/diamond motif.
- `rough-slashes.svg`: scene resolve and structural collision.

Motifs are intentionally abstract and scalable. They should be selected from the song-style analysis and timeline `motifPack`, not placed as generic decoration.
# Built-in kinetic motifs

All motifs are original SVG system assets and can be recolored in a renderer. They are semantic objects, not emoji substitutes.

- `coin-seal.svg`: prosperity, stakes, a sealed wager.
- `mahjong-tile.svg`: table culture, chance, a physical game object.
- `paper-rip.svg`: analog scene hand-off / collage transition.
- `sprayburst-red.svg`, `faceted-gem.svg`, `rough-slashes.svg`: legacy code/stencil pack.
